Introdução ao MVC

O padrão MVC (Model-view-Controller) é uma arquitetura de software ultilizado no desenvolvimento de aplicação para separar a lógica de negócios, a interface do usuario e o controle dos dados.

    Componentes do MVC

  • Model (Modelo) - Representa os dados e a lógica da aplicação.
  • View(visão) - Responsavel pela interface com o usuário e exibição dos dados
  • Controller (Controlador) - Intermedía a comunicação entre Model e View.
