Model (Modelo) no MVC

O Model Representa a camada de dados da aplicação. Ele representa a lógica de negócios,s regras e a interação com o banco de dados.

Pincipais Funcões do Model

  • Armazena e garencia os dados.
  • Executar regras de negócios
  • Interage com o banco de dados.

Exemplo

Imagine um sistema de usuário onde o medel contém as informações como nome, email e senha, além das regras de validação.
